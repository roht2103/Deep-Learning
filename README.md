# Deep Learning Course

This repository contains lab assignments and projects for the Deep Learning course (6th Semester B.Tech).

## Course Structure

### Lab Assignments

#### Lab Assignment 1: Neural Network from Scratch

Implementation of a basic feedforward neural network using only NumPy to understand the fundamental concepts of:

- Forward propagation
- Backpropagation
- Gradient descent
- Activation functions (Sigmoid)

**Topics Covered**:

- Neural network architecture design
- Weight initialization
- Activation functions and their derivatives
- Training loop implementation
- Mean Squared Error loss
